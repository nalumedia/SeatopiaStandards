Product compasses

Each compass rose is a four-axis radar. Fully filled = 100 on every dimension. The shape tells the story — a lopsided polygon points directly at what to improve.

Formula · auditable & defensible

The Omega Compass is a weighted average of four sub-scores (each 0–100). Every Seatopia product has its sub-scores computed from lab data, not marketing estimates. Thresholds are anchored to published clinical / regulatory references so the claim survives scrutiny.

Compass Score = (w_D · Density) + (w_B · Balance) + (w_P · Purity) + (w_Pr · Protein) where default weights: w_D=0.35, w_B=0.20, w_P=0.30, w_Pr=0.15 Density = min(100, EPA+DHA[mg/100g] / 15) → 1,500 mg = 100; anchored to AHA therapeutic range (1–2 g EPA+DHA/day) Balance = max(0, min(100, 100 + 76.7 · log10(n3/n6))) → ratio 1:1 = 100; ratio 1:20 (Western diet) = 0 → source: Simopoulos 2002/2008, AJCN dietary ratio benchmarks Purity = Hg_score (0–60) + MP_score (0–40) Hg_score = 60 · max(0, 1 - Hg_ppm / 0.23) → 0 ppm = 60; 0.15 (EPA Best Choices) = 21; 0.23 (Good Choices) = 0 MP_score = 40 if 0/6 detected, 20 if not tested, 0 if any detection → source: EPA/FDA 2021 advisory; EMSL 6-polymer FTIR panel, LOD 1 microparticle Protein = min(100, protein[g/100g] · 4) → 25 g = 100; anchored to USDA "high protein" threshold (20%+ DV per serving)

All four formulas are linear or log-linear, deliberately — so a 10% data change produces a predictable score change. No black-box weighting, no ML, no opaque weighting. That's the point.

Why these four dimensions (and not more)?

The field-tested rule for branded wellness metrics is 3–5 sub-scores. Fewer and the score loses nuance; more and users stop reading. Oura Sleep Score uses 7 but collapses them into 3 visible tiers. Whoop Recovery uses 4. Apgar uses 5. We chose four because they map cleanly to what Seatopia provably differentiates on: density (genetics + feed), balance (feed chemistry), purity (sourcing + testing), protein (species choice). Contaminant sub-categories (PCBs, dioxins, antibiotics, selenium) roll into Purity as we add assays — the compass doesn't need to grow more axes.

Score tiers

Apex 90+   Flagship — the "this is why Seatopia exists" story. Leads hero images.
Gold 75–89   Strong across the board. Core assortment.
Verified 60–74   Meets Seatopia Standard; typical farmed alternative for this species.
Baseline <60   Passes testing but the category itself is low-density (shrimp, scallops, white fish). Marketed on purity + protein, not omega volume.
